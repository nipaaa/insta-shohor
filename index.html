How does JavaScript Work..??

Javascript an interpreted programming language.JavaScript is a single-threaded programming language, which means it has a single Call Stack. It can handle one task at a time.It uses dynamic typing.And it is verified at the runtime.JavaScript has no compilation step. An interpreter in the browser reads over the JavaScript code, interprets each line, and runs it.

Different between Local Storage and Session Storage
Local Storage

1. The capacity of local storage is 10mb .
2. Local storage data can be asccesible from any window.
3. Local storage data never expires.

Session Storage

1. The capacity of session storage is 5mb
2. Session storage data can be accessible from same tab.
3. Session storage data expires on tab close.
